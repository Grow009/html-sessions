# 📘 HTML Training Sessions

This repository/document contains the structure and content of HTML training sessions conducted to help beginners understand the fundamentals of web development.

---

## 📅 Session Overview

### 🔹 HTML Core Sessions (6 Sessions)

1. **Introduction to HTML**
   - What is HTML?
   - Basic structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
   - First webpage
   - How Website Load on the Browser.

2. **HTML Elements, Tags and Attributes & Their Types (Basic to Advance)**
   - Headings, paragraphs
   - Formatting tags (`<b>`, `<i>`, `<strong>`, etc.)

3. **Links & Images and Their Attribute**
   - Anchor tags
   - Image tags
   - Attributes

4. **Lists & Tables & Their Types (Basic to Advance)**
   - Ordered & unordered lists
   - Table structure (`<table>`, `<tr>`, `<td>`)

5. **Forms in HTML (Basic to Advance)**
   - Input types
   - Forms, labels, buttons
   - Basic form validation

6. **Semantic HTML**
   - Semantic tags (`<header>`, `<footer>`, `<section>`, etc.)
   - Importance of clean structure

---

## 🔁 Revision Sessions (2 Sessions)

### 🧠 Session 7: Revision
- Recap of all HTML concepts
- Practice exercises
- Doubt solving

### 🎤 Session 8: Mock Interview
- HTML-based interview questions
- Practical coding tasks
- Feedback and improvement tips

---

## 🎯 Objectives

- Build a strong foundation in HTML
- Understand real-world usage of HTML
- Prepare learners for interviews
- Encourage hands-on practice

---

## 🛠️ Tools Used

- Text Editor (VS Code / Notepad)
- Web Browser (Chrome)

---

## 📌 Outcome

By the end of these sessions, learners are able to:
- Create structured web pages
- Use different HTML elements effectively
- Build basic forms and layouts
- Answer HTML interview questions confidently
